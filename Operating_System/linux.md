### screen 명령어
- `screen -S 'screen name'`: Create new screen session
- `screen -ls`: Show screen list
- `screen -r 'screen name'`: detached되어있는 스크린에 attached(접속) 한다
- `Ctrl+a, d`: Detached from screen session

### mutt 명령어
- `mutt`: 서버 내부 메일 시스템
